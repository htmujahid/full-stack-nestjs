About crude

A minimal full-stack foundation for developers who want to ship fast without the boilerplate.

Our mission

We built crude to reduce the friction between idea and deployment. Every project starts with auth, database, and a solid UI — so you can focus on what makes your product unique.

The stack

NestJS 11, React 19, Vite 7. TypeScript end-to-end. shadcn UI for accessible, customizable components. TypeORM for the database. Passport for auth — password, Google, email, phone, 2FA.

Open source

No vendor lock-in. Host it wherever you want. Customize and extend freely.
